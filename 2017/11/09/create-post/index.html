npm 升级到5.4.1后遇到的权限问题

发表于 2017-11-09 | 分类于 npm |

升级后遇到
npm ERR! Please try running this command again as root/Administrator

尝试了很多种方法，都无法解决
最终步骤：
1、先把node_module删掉
2、npm install npm @4.6.1 -g 进行降级
3、npm install npm @latest -g 进行升级到最新的 现在是5.4.2
4、npm cache clean --force 清理缓存
5、npm install 等待安装
后面再安装其他包就不报错了
